Flexible Lighting Switching using a mobile app . 2020-10-27

Time-controlled switching of devices is easily possible using a web / browser dialog on the smartphone, tablet or PC.

No more is required for setup, operation, log output and documentation is a browser.

piSchedule is an excellent tool to switch the lighting of your apartment / house even when you are away . This simulates activities in the house. A daily schedule can change the switching times with sunrise / sunset and all times can also be varied by random values.

A detailed description of operation and installation can be found in the documentation:


  • Overview / Usage and Setup
  • Schedule Job Edit
  • Timeline / Graphical View of switch times
  • Day Plan / Job Examples
  • Job Features
